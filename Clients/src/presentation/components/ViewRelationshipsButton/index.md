# View Relationships Button

The View Relationships Button provides quick access to the Entity Graph from any entity row in a table.

## Overview

This button appears as a branch icon in table rows for models, vendors, risks, and use cases. Clicking it opens the Entity Graph modal focused on that specific entity.

## Appearance

- **Icon**: Branch/tree icon
- **Color**: Grey by default, green on hover
- **Tooltip**: "View relationships"

## Behavior

When clicked:
1. The Entity Graph modal opens
2. The graph centers on the clicked entity
3. The entity is highlighted with an orange glow
4. Related entity types are automatically displayed

## Where it appears

The button is available in:
- Model inventory table rows
- Vendor table rows
- Risk management table rows
- Use case table rows

## Tips

- Hover over the button to see the tooltip
- The button is small to avoid cluttering the table interface
- Use this to quickly understand an entity's connections without navigating away from your current view
