Settings

The Settings page provides comprehensive management of your account, organization, and team. It allows you to configure personal information, security settings, team members, and organizational details. Use this page to maintain your profile, manage access controls, and configure your organization's settings.

System Overview

The Settings page is organized into four main tabs, each serving specific configuration needs. The tab-based interface provides easy navigation between different settings categories, ensuring a clean and organized user experience. Each tab contains relevant functionality based on your role and permissions.

Settings Tabs

Profile Tab

The Profile tab allows you to manage your personal account information and account deletion:

  • Personal Information: Update your first name, last name, and view your email address
  • Email Management: View your current email (cannot be changed for security reasons)
  • Account Deletion: Permanently delete your account and all associated data
  • Form Validation: Real-time validation for name fields with error messages
  • Change Tracking: Save button only enabled when changes are made
Password Tab

The Password tab provides secure password management functionality:

  • Password Change: Update your current password with security validation
  • Current Password Verification: Enter your current password for security
  • New Password Requirements: Strong password validation with complexity rules
  • Password Confirmation: Confirm new password to prevent typos
  • Security Validation: Comprehensive password strength checking
Team Tab

The Team tab manages team members, roles, and invitations (Admin and Editor roles only):

  • Team Member Management: View and manage all team members
  • Role Assignment: Assign and modify user roles within the organization
  • User Invitations: Send invitations to new team members
  • Member Deletion: Remove team members from the organization
  • Role Filtering: Filter team members by their assigned roles
  • Pagination: Navigate through large team member lists
Organization Tab

The Organization tab manages organizational settings and branding:

  • Organization Name: Set and update your organization's name
  • Logo Management: Upload, update, and remove organization logos
  • Organization Creation: Create new organizations (if permitted)
  • Branding Control: Manage visual identity and branding elements
  • File Upload: Support for various image formats with size validation

Password Management Features

Password Requirements

The system enforces strong password requirements for security:

  • Minimum Length: At least 8 characters required
  • Maximum Length: Up to 128 characters allowed
  • Uppercase Letters: Must contain at least one uppercase letter
  • Lowercase Letters: Must contain at least one lowercase letter
  • Numbers: Must contain at least one number
  • Special Characters: Must contain at least one special character
Password Change Process
  • Current Password Verification: Enter existing password for security
  • New Password Entry: Enter new password with real-time validation
  • Password Confirmation: Confirm new password to prevent errors
  • Confirmation Dialog: Double confirmation before password change
  • Success Feedback: Clear confirmation of successful password update
Security Features
  • Real-time Validation: Immediate feedback on password strength
  • Match Verification: Confirmation password must match new password

  • Secure Transmission: Password data encrypted during transmission

  • Loading Problems: Clear browser cache and refresh the page
